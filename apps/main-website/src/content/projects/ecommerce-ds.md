## Project Overview
A design system built to scale. It provides consistent UI components and design patterns for complex e-commerce flows.

### Key Features
- **Multi-Platform Tokens**: Exports tokens to CSS, SCSS, and JS.
- **Storybook Documentation**: Comprehensive guide for developers and designers.
- **Accessible Components**: WCAG-compliant UI library.
