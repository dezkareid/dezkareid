## Project Overview
This project serves as the foundation for modern web portfolios. It emphasizes speed, accessibility, and maintainability.

### Key Features
- **Design Tokens Integration**: Unified styling via a central tokens package.
- **Astro Components**: Modular and performant UI structure.
- **SEO Optimized**: Pre-configured meta tags and schema data.
