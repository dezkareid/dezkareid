## Project Overview

AI agents work better when they understand the company context they operate in. This project provides MCP (Model Context Protocol) plugins that expose architecture principles, company outcomes, and team conventions directly to AI agents during development sessions.

### Key Features

- **Company context plugin**: Surfaces architecture principles and team outcomes to agents.
- **MCP-compatible**: Integrates with Claude Code and other MCP-aware tools.
- **Extensible**: Designed to support additional context providers as plugins.
