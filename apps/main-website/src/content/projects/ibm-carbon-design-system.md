## Project Overview

Carbon is IBM's open-source design system used across hundreds of enterprise products worldwide. I contributed improvements to React components, accessibility fixes, and documentation to help the broader community.

### Key Contributions

- **Component improvements**: Fixed edge cases and improved API consistency for several core components.
- **Accessibility**: Resolved keyboard navigation and ARIA attribute issues to better meet WCAG standards.
- **Documentation**: Improved Storybook stories and usage examples to make components easier to adopt.
